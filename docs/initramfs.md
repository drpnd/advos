# Initramfs

## Format

```
 0        1        2        3        4        5        6        7
+--------+--------+--------+--------+--------+--------+--------+--------+
| Filename                                                              |
+                                                              +--------+
|                                                              | Attr   |
+--------+--------+--------+--------+--------+--------+--------+--------+
| Offset                                                                |
+--------+--------+--------+--------+--------+--------+--------+--------+
| Size                                                                  |
+--------+--------+--------+--------+--------+--------+--------+--------+
```

* Filename: Name of the file
* Attr: Attribute
* Offset: Offset to the file in bytes
* Size: Size of the file in bytes

### Attribute
```
 0 1 2 3 4 5 6 7 
+-+-+-+-+-+-+-+-+
| Reserved      |
+-+-+-+-+-+-+-+-+
```
